# Bedrock And Beyond Add-On

## For Player
This Add-On is still under development phase, and not ready to play yet! [Join our Discord](https://discord.blockstate.team) to get further update!
## For Developer
**Step 1:** Clone the repo to your Desktop or Document

**Step 2:** Go to your **`development_behavior_packs`** folder, then open **Command Promt** NOT Powershell, paste this command into it: **`mklink /J ".\Bedrock And Beyond" "C:\Users\YourUserName\Desktop\Bedrock And Beyond\RP"`** This will create a symbolic link that Minecraft will use the content in the Desktop as the source of truth. So you can just make the changes in one directory only.

**Step 3:** Do the same for the Behavior Pack. And you're ready to go.
